# Student Management System

A **simple desktop application** built using Python and CustomTkinter for managing student information. It allows users to add, edit, delete, search, and update student records, all stored in a JSON file. The app also supports dark/light theme switching.

---

## Features

* Add Student: Input Name, Roll No, Class, and Course to add a student.
* Edit & Update: Select a student to edit details and save changes.
* Delete: Remove student records from the system.
* Search: Search by student name or roll number.
* Refresh Table: Reload the table to display all students.
* Clear Fields: Clear input fields quickly.
* Dark/Light Mode: Toggle between dark and light themes.
* Persistent Storage: Student records are saved in students.json.

---

## Running on Kali Linux (Recommended)

**If you are using Kali Linux, it is recommended to run the project inside a virtual environment.

### 1 Clone the Repository

```bash
git clone https://github.com/Hardik7219/Student_managment_system
cd Student_managment_system
```

### 2 Create Virtual Environment

```bash
python3 -m venv venv
```

### 3 Activate Virtual Environment

```bash
source venv/bin/activate
```

### 4 Install Dependencies

```bash
pip install -r requirements.txt
```

### 5 Run The Application

```bash
python3 studentManegament.py
```
